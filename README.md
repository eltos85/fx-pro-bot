# fx-pro-bot — Сканер-советник с автопроверкой

Бот сканирует валютные пары, металлы и нефть, находит сигналы по пересечению скользящих средних и автоматически проверяет свою точность через 15, 30 и 60 минут.

**Без сделок, без логина к брокеру** — только советы и статистика.

## Что сканирует

| Категория | Инструменты |
|-----------|-------------|
| Валюты | EUR/USD, GBP/USD, USD/JPY, AUD/USD, USD/CAD, EUR/GBP |
| Металлы | Золото (XAU), Серебро (XAG) |
| Нефть | WTI, Brent |

## Быстрый старт (Docker)

```bash
cp .env.example .env
# отредактируйте .env при необходимости
docker compose up -d
docker logs -f fx-pro-bot-advisor-1
```

## Установка без Docker

```bash
python -m venv .venv
source .venv/bin/activate
pip install ".[quotes]"
fx-pro-bot          # запуск сканера
```

## Настройки (переменные окружения)

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| `SCAN_SYMBOLS` | все 10 инструментов | Тикеры Yahoo через запятую |
| `YFINANCE_INTERVAL` | `5m` | Интервал свечей |
| `YFINANCE_PERIOD` | `1mo` | Глубина истории (нужно ≥205 H1 баров для HTF EMA200) |
| `POLL_INTERVAL_SEC` | `300` | Частота сканирования (секунды) |
| `VERIFY_HORIZONS` | `15,30,60` | Через сколько минут проверять (через запятую) |
| `LOG_LEVEL` | `INFO` | Уровень логирования |

## Статистика (CLI)

```bash
fx-pro-stats report       # полный отчёт: win-rate, профит по инструментам
fx-pro-stats list -n 10   # последние 10 сигналов с результатами проверки
fx-pro-stats mark <id> right --notes "комментарий"   # ручная оценка
```

## Как работает автопроверка

1. Бот даёт сигнал (LONG/SHORT) и записывает цену
2. Через 15, 30 и 60 минут проверяет текущую цену
3. Считает профит в пунктах:
   - LONG: `(цена_сейчас − цена_сигнала) / размер_пункта`
   - SHORT: `(цена_сигнала − цена_сейчас) / размер_пункта`
4. Если профит > 0 — сигнал верный, иначе — ошибочный
