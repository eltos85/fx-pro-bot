### Итоговая оценка

**Статус: направление верное, но недостаточно данных для окончательных выводов.**

Правила сверки (`sample-size.mdc`):
- ≥100 сделок на связку — **выполнено только для агрегата** (104); по стратегии
  scalp_vwap n=72, scalp_statarb n=24 — **недовыборка**.
- ≥2 недели — **не выполнено** (~2.8 дня).
- p-value WR агрегат: 36.1% vs 51.0% на выборках 316/104 — z≈2.7, p≈0.006 —
  статистически значимо (только для агрегата, не для стратегий).
- R:R: avg_win/|avg_loss| = 0.85 (БЫЛО) vs 0.48 (СТАЛО). R:R **ухудшился**,
  компенсируется ростом WR.

**Решение: ничего не трогаем, продолжаем собирать данные.** Следующий срез через
3-4 дня (~2026-04-22), когда СТАЛО достигнет ~200-250 сделок и scalp_vwap
наберёт ≥100 сделок для проверки гипотезы о его убыточности.
